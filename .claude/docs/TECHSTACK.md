# 技術棧 (Tech Stack)

> 由 `/concept` 維護

---

## 總覽

此專案為輕量級的設定同步工具，主要使用 PowerShell 腳本實作，不需要複雜的框架或依賴。

---

## 核心技術

### 腳本語言

| 技術 | 版本 | 用途 |
|------|------|------|
| PowerShell | 5.1+ / 7+ | 主要腳本語言 |

### 版本控制

| 技術 | 用途 |
|------|------|
| Git | 設定檔版本控制 |
| GitHub | 遠端儲存庫 |

---

## 不需要的技術

此專案刻意保持簡單，**不需要**以下技術：

- 程式語言：Python、Node.js 等
- 套件管理：npm、pip 等
- 建構工具：無需編譯或打包
- 資料庫：無
- 容器化：無

---

## 開發環境

### 必要工具

| 工具 | 說明 |
|------|------|
| Windows Terminal | 終端機（也是同步目標） |
| Claude Code CLI | 開發輔助 |
| Git | 版本控制 |
| VSCode / 任意編輯器 | 編輯腳本 |

### 目標環境

| 項目 | 規格 |
|------|------|
| 作業系統 | Windows 10/11 |
| PowerShell | 5.1+（內建）或 7+（推薦） |
| 權限 | 可能需要管理員權限（設定環境變數） |

---

## 檔案格式

| 格式 | 用途 |
|------|------|
| `.ps1` | PowerShell 腳本 |
| `.json` | 設定檔（Claude、Terminal、MCP） |
| `.md` | 文件 |
| `.env` | 環境變數範本（不含敏感資料） |

---

## 選擇原因

| 決策 | 原因 |
|------|------|
| PowerShell | Windows 原生支援，無需安裝額外依賴 |
| 純腳本方案 | 簡單、透明、易於維護與理解 |
| Git 同步 | 版本控制、變更追蹤、多裝置同步 |
| 不用 Chocolatey/Scoop | 避免額外依賴，保持最小化 |
