=====================================
  Claude Code 開發環境 - 快速設定指南
=====================================

【使用方法】

  1. 右鍵點擊「一鍵安裝.bat」
  2. 選擇「以系統管理員身分執行」
  3. 按任意鍵開始安裝
  4. 等待安裝完成
  5. 關閉視窗，開啟新的終端機
  6. 輸入 claude 開始使用


【安裝內容】

  - Git：版本控制工具
  - Visual Studio Code：程式碼編輯器
  - Node.js LTS：JavaScript 執行環境
  - Claude Code CLI：Claude 的命令列工具


【換電腦使用】

  這個資料夾可以直接複製到任何電腦使用：
  1. 將整個「初始環境設定」資料夾複製到新電腦
  2. 右鍵執行「一鍵安裝.bat」即可

  適用於：
  - 新電腦
  - 重灌系統後
  - 任何 Windows 10/11 電腦


【首次使用 Claude Code】

  1. 開啟終端機（PowerShell 或 CMD）
  2. 輸入 claude 並按 Enter
  3. 按照提示登入你的 Anthropic 帳號
  4. 完成認證後即可開始使用


【常用 Claude Code 指令】

  claude              啟動互動模式
  claude "問題"       直接問問題
  claude /help        查看幫助


【系統需求】

  - Windows 10 (版本 1809+) 或 Windows 11
  - 需要有 winget 套件管理器（Windows 內建）
  - 網路連線


【疑難排解】

Q: 執行 claude 顯示「不是內部或外部命令」
A: 請關閉終端機重新開啟，讓環境變數生效

Q: winget 不可用
A: 請從 Microsoft Store 安裝「應用程式安裝程式」(App Installer)

Q: npm 指令找不到
A: Node.js 安裝後需要重新開啟終端機

Q: 雙擊 bat 檔沒反應
A: 請改用「右鍵 > 以系統管理員身分執行」
